# 🦇 Batman Medal 3D

# 🌟 Task Instructions:

For the second mechanics task: Choose any image from the internet and measure it yourself. For example, I chose the Batman logo, and I will draw it, cut a 4mm circle for the keyholes, then extrude it by 2mm, then export it in STL format and upload it to GitHup with a link to the OnShape design.

# ✨ The solution:

A premium 3D-printable Batman medal with detailed engravings and a professional design.

---

## ✨ Project Highlights

🦇 **High-Quality Design** - Detailed bat silhouette with engraved patterns  
⭕ **Hanging Hole** - 4mm diameter for chains or ribbons  
📏 **Perfect Size** - 50mm × 28mm × 4mm (ideal for medals)  
🎨 **Professional Finish** - Smooth curves and clean geometry  
⚡ **Ready to Print** - Optimized STL file for FDM & Resin
